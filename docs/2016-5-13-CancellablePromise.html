CancellablePromise

new CancellablePromise(executorFn) → {CancellablePromise}

Provide a Promise-style interface but expose a .cancel() method on the returned "Promise"-like object that will prevent any registered success callbacks from executing. (Failure callbacks will execute as normal.)

Parameters:
Name Type Description
executorFn function

A normal Promise executor function, accepting "resolve" and "reject" callback function properties.
