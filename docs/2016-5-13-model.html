(static) primitive(id, primitive) → {Object}

A setter for primitives based on IDs. To prevent memory leaks, primitives do not directly store references to other primitives. Instead, they store IDs and use model.lookup as a lookup. When a new primitive is created, it calls this function to store itself in the model.
