Manipulators

Represents the Manipulators, a Vega data transformation operator.

Constructor

new Manipulators(graph) → {Object}

The Manipulators transform is a base class that should be subclassed for each mark type. It provides the base transform methods that compute the data to drive the manipulator mark specifications when in a particular manipulator mode for a selected visualization mark item.
