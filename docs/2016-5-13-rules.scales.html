(private, static) parse(def) → {Object}

Parse a Vega scale definition (produced by Vega-Lite) and return an object that mimics Lyra's Scale primitive. Note: this does not construct a Lyra Scale primitive, but instead produces an object to compare existing Scale primitives against. We map from Vega scale DataRefs to Lyra Primitive IDs, and account for Vega-Lite idiosyncracies such as hardcoded ranges and band sizes.
