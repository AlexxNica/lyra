Scale

A Lyra Scale Primitive.

Constructor

new Scale(name, type, domain, range)

The Scale Primitive corresponds to a definition for a Vega scale.

Parameters:
Name Type Description
name string

The initial name of the scale. It may be renamed to prevent scale name collisions.
type string

The scale type (e.g., ordinal, linear, etc.).
domain *

The scale domain (values in data space).
range *

The scale range (values in visual space, e.g., width).
Properties:
Name Type Description
_domain Array.<number>

An array of primitive IDs if the scale's domain is a set of fields (a Vega DataRef).
_range Array.<number>

An array of primitive IDs if the scale's range is a set of fields (a Vega DataRef).
